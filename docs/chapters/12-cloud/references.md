# References: Cloud Computing and Infrastructure

1. [Cloud computing](https://en.wikipedia.org/wiki/Cloud_computing) - Wikipedia - Comprehensive overview of IaaS, PaaS, SaaS service models and public/private/hybrid deployment models. Anchors the chapter's cloud framing.

2. [Serverless computing](https://en.wikipedia.org/wiki/Serverless_computing) - Wikipedia - Detailed treatment of FaaS and serverless patterns, vendors, and tradeoffs. Foundation for the modern cloud section.

3. [Vendor lock-in](https://en.wikipedia.org/wiki/Vendor_lock-in) - Wikipedia - Coverage of vendor lock-in mechanisms, switching costs, and mitigation strategies. Directly relevant to the chapter's cloud-strategy content.

4. Cloud Computing: Concepts, Technology, and Architecture (2nd Edition) - Thomas Erl, Eric Barceló Monroy - Pearson - Comprehensive vendor-neutral textbook on cloud architectures, deployment models, and patterns; matches this chapter's level.

5. Cloud FinOps (2nd Edition) - J.R. Storment and Mike Fuller - O'Reilly - The standard reference for FinOps practice, cost management, and the CapEx-vs-OpEx model shift; directly supports the chapter's financial content.

6. [NIST SP 800-145: The NIST Definition of Cloud Computing](https://csrc.nist.gov/publications/detail/sp/800-145/final) - NIST - The authoritative definition of the five cloud characteristics, three service models, and four deployment models that this chapter teaches.

7. [AWS Cloud Adoption Framework](https://aws.amazon.com/cloud-adoption-framework/) - Amazon Web Services - Major-vendor framework for planning cloud migrations, including the Six Rs decision tree referenced in the chapter MicroSim.

8. [Microsoft Cloud Adoption Framework](https://learn.microsoft.com/en-us/azure/cloud-adoption-framework/) - Microsoft Learn - Comprehensive guidance on cloud strategy, migration, and governance with practical templates. Excellent complement to the AWS framework.

9. [Shared Responsibility Model](https://aws.amazon.com/compliance/shared-responsibility-model/) - Amazon Web Services - Authoritative explanation of customer-vs-provider responsibilities across IaaS, PaaS, and SaaS. Directly supports the chapter's shared-responsibility MicroSim.

10. [FinOps Foundation Framework](https://www.finops.org/framework/) - FinOps Foundation - Open framework for cloud financial management with maturity models, capabilities, and personas. Reinforces the chapter's cost-management content.
