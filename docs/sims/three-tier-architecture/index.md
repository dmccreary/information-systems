## About This MicroSim

A vertical three-tier architecture (Presentation → Application → Data) with a **Trace a Request** animation. Pick a workload (product page, loan application, monthly report) and watch a request travel down through the tiers and the response travel back up.

## Lesson Plan

### Learning Objectives

By the end of this activity, students will be able to:

1. Identify which tier owns a given responsibility
2. Trace a request through three tiers and a response back
