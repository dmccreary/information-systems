# Mascot Image Prompts: Iris the Hummingbird

Pedagogical agent for the **Information Systems** intelligent textbook. Iris embodies the qualities the IS profession prizes: efficiency, agility, precision, and rapid iteration.

## Character Reference Sheet

- **Name:** Iris
- **Species:** Ruby-throated hummingbird
- **Personality:** Quick, precise, curious, encouraging
- **Catchphrase:** "Fast, focused, and on the wing!"
- **Visual:** Iridescent emerald-green back and crown, ruby-magenta throat patch (gorget), pale cream belly, small round wire-rim glasses. Big friendly dark eyes. Long slender black beak. Compact, rounded body proportions — slightly stylized for a friendly mascot read.
- **Art style:** Modern flat vector illustration, clean lines, soft cel-shading, transparent background, suitable for embedding in educational content.
- **Sizing:** Generate at 1024x1024. Subject should fill the frame (minimal padding). Trim transparent padding after generation using `trim-padding-from-image.py`.

## How to Use These Prompts

1. Open your AI image generator (ChatGPT/DALL-E, Midjourney, Stable Diffusion, etc.).
2. Paste **one full prompt below** at a time — each prompt is self-contained, so you do not need to repeat the character sheet.
3. Save the result with the exact filename listed at the end of the prompt into `docs/img/mascot/`.
4. After all seven images are saved, run the trim script (commands at the bottom of this file) to remove excess transparent padding.

## After Generation: Trim Excess Padding

AI image generators usually add transparent padding around the subject, which makes the mascot render too small inside admonitions. Run the trim script on each saved file:

```bash
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/neutral.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/welcome.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/thinking.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/tip.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/warning.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/encouraging.png
python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/celebration.png
```

## Required Specifications

- **Format:** PNG with transparent background
- **Display dimensions:** 200x200 to 400x400 pixels (after trim)
- **File size:** Under 100KB per image for web performance
- **Background:** Fully transparent — no white, black, or checkered fills

## Why a Hummingbird?

The hummingbird is a near-perfect metaphor for the modern Information Systems professional:

- **Tiny but powerful** — small organizations and small teams can move IS-scale workloads with the right architecture.
- **Remarkable precision in flight** — IS work demands exactness in data, security, and process.
- **Energetic and rapid** — the field rewards agility, fast iteration, and continuous delivery.
- **Hovers between sources** — IS sits at the seam of people, processes, and technology, moving fluidly between them.
- **Iridescent** — the field reflects whatever organizational light shines on it: business, engineering, governance, AI.
